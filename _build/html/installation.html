Installation

Prerequisites

  • Python 3.10+ (tested on 3.11)

  • A working Python environment (venv or conda)

Install the Package

Clone the repository and run:

bash install.sh

This installs the amber package in your active environment.

Optional extras

  • To use notebooks in the docs: pip install -r docs/requirements.txt

  • For TOPAS/TOPAS-nBio dose import, follow your local installation (not part of this repo).
